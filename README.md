2.make sure that you are in the proper directory (quishThis) by typing in the command cd quishThis
3.type in the command -> npm install
4.type in the command -> npm install expo-camera expo-barcode-scanner axios react-native-url-polyfill expo-font
5.type in the command -> npm start

6.scan the QR code with your camera and then scan a QR code through the program.
